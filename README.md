# AI Hediye Danışmanı

AI Hediye Danışmanı, kullanıcının girdiği bilgilere göre yapay zekâ destekli 3 yaratıcı hediye önerisi sunan basit bir web uygulamasıdır.

## Özellikler
- Kullanıcıdan hediye bilgilerini alır
- İlgi alanı, bütçe ve hediye tarzına göre öneri üretir
- Google Gemini API kullanır
- Streamlit ile basit ve temiz bir arayüz sunar

## Kullanılan Teknolojiler
- Python
- Streamlit
- Google Gemini API
- python-dotenv
